<h2>📌 DESCRIPCIÓN EXAGERADAMENTE DETALLADA</h2>
<p style="font-size:1.3rem;">
    <strong>CHANDE‑RUN</strong> no es solo un juego: es una <span class="glow">experiencia sensorial</span> que combina la precisión de un saltador de élite, 
    la rapidez de un cheetah con cafeína y la sed de un caminante del desierto. 
    Controlas a un intrépido <strong>stickman (por ahora un rectángulo rojo con aspiraciones)</strong> que debe sobrevivir el mayor tiempo posible 
    mientras la velocidad se vuelve endemoniada. Olvídate de los aburridos contadores de nivel: aquí lo que importa son las 
    <strong>🥤 LATAS 🥤</strong>. Cada lata te acerca a comprar el legendario poder del <strong>SEGUNDO INTENTO</strong> (revivir tras morir, solo 50 latas).
</p>

<p style="font-size:1.3rem;">
    La física del salto es <strong>dinámica y realista</strong> (vale, un poco exagerada): cuanto más tiempo mantengas la barra espaciadora, 
    más alto volará tu stickman, desde un mini saltito hasta un brinco capaz de superar rascacielos (o al menos una caja alta). 
    Pero cuidado: las <span style="color:#ff6666; font-weight:bold;">SIERRAS GIRATORIAS</span> te rebanarán sin piedad si las tocas, a menos que tengas el poder de revivir.
</p>

<h2>🔥 CARACTERÍSTICAS QUE TE DEJARÁN HELADO</h2>
<ul>
    <li><strong>🏃 Runner infinito procedural:</strong> el escenario (carretera, cajas y sierras) se genera sin fin, cada partida es única.</li>
    <li><strong>🥤 Economía de LATAS:</strong> la moneda del juego son latas de refresco. Recolecta, gasta en la tienda, siéntete rico.</li>
    <li><strong>🔄 Salto con carga:</strong> mantén ESPACIO para saltar más alto. Suéltalo y vuela. (Técnica avanzada: salto milimétrico para esquivar).</li>
    <li><strong>📦 Cajas que NO MATAN si pasas por encima:</strong> lógica de colisión inteligente: si tu rectángulo está por arriba de la caja, sobrevives. Si chocas de frente, GAME OVER (salvo power-up).</li>
    <li><strong>⚙️ Sierra mortal:</strong> un enemigo simple pero letal. Su diseño de dientes te recordará que no debes confiarte.</li>
    <li><strong>🛒 Tienda con power-up “Segundo intento”:</strong> gasta 50 latas y tendrás una vida extra automática al morir. Se activa una sola vez.</li>
    <li><strong>🎛️ HUD minimalista:</strong> solo ves latas acumuladas y puntuación. Nada de XP, niveles ni números molestos. (Botón SALIR abajo a la derecha).</li>
    <li><strong>⏸️ PAUSA con menú:</strong> ESC pausa, Q te lleva al menú sin perder la partida (queda congelada).</li>
    <li><strong>🌌 Fondo personalizable:</strong> usa tu propia imagen 1920x1080 en <code>assets/fondo.jpg</code>. Si no, fondo azul de emergencia.</li>
    <li><strong>🧃 Sprite de lata opcional:</strong> coloca <code>assets/refresco.png</code> (32x32, fondo transparente) y se dibujará tu lata. Si no, un rectángulo rojo dirá “LATA”.</li>
</ul>

<h2>🎮 CÓMO JUGAR (MANUAL DEL EXPERTO EN REFRESCOS)</h2>
<ol>
    <li><strong>Menú principal:</strong> presiona <code>ESPACIO</code> para comenzar, <code>T</code> para tienda, <code>ESC</code> para salir del juego.</li>
    <li><strong>En carrera:</strong>
        <ul>
            <li><code>ESPACIO (mantener)</code> – cargar salto / soltar para saltar.</li>
            <li><code>FLECHA ABAJO</code> o <code>S</code> – agacharse (útil para esquivar sierras bajas? en realidad las sierras siempre matan, pero las cajas se evitan saltando).</li>
            <li><code>ESC</code> – pausar (aparece menú: ESC reanuda, Q va al menú).</li>
            <li><strong>Botón SALIR</strong> (esquina inferior derecha) – clic para pausar.</li>
        </ul>
    </li>
    <li><strong>Objetos:</strong>
        <ul>
            <li><span style="color:#ffd966;">🥤 Latas</span> – recógelas para incrementar tu contador.</li>
            <li><span style="color:#b0b0b0;">⚙️ Sierras</span> – muerte instantánea (a menos que tengas Segundo Intento).</li>
            <li><span style="color:#b85e00;">📦 Cajas</span> – si caes justo encima no pasa nada; si chocas de lado, mueres.</li>
        </ul>
    </li>
    <li><strong>Tienda:</strong> con <code>T</code> accedes, y con <code>1</code> compras “segundo intento” si tienes 50 latas. El poder se activa automáticamente la próxima vez que mueras.</li>
</ol>

<h2>📦 INSTALACIÓN (para valientes que quieran ejecutar en su máquina)</h2>
<pre style="background:#1c1f2b; color:#b0ffb0; padding:30px; border-radius:40px; border:3px solid #7a4edb;">
1. Clona este repositorio (o descarga el ZIP)
   👉 git clone https://github.com/TU-USUARIO/Chande-Run.git

2. Entra a la carpeta
   👉 cd Chande-Run

3. Asegúrate de tener Python 3.10+ y pygame
   👉 pip install pygame

4. Coloca tu fondo en assets/fondo.jpg (opcional)
   y tu sprite de lata en assets/refresco.png (opcional)

5. Ejecuta la locura
   👉 python main.py
</pre>

<h2>⚙️ TECLADO / CONTROLES (CHULETA RÁPIDA)</h2>
<ul>
    <li><code>ESPACIO</code> (mantener) → cargar salto / soltar → saltar</li>
    <li><code>↓</code> / <code>S</code> → agacharse</li>
    <li><code>ESC</code> → pausa (si estás jugando) / volver al menú si estás en pausa</li>
    <li><code>T</code> → abrir tienda (solo en menú)</li>
    <li><code>1</code> → comprar power-up en tienda</li>
    <li><code>Q</code> → salir al menú desde la pausa</li>
    <li><strong>Ratón:</strong> clic en botón SALIR (abajo derecha) → pausa</li>
</ul>

<h2>🧠 FILOSOFÍA DE DISEÑO (el manifiesto Chande)</h2>
<p style="font-size:1.2rem; background:#242837; border-radius: 60px; padding:25px;">
    Lejos de los juegos hipercomplejos llenos de menús y números, <strong>Chande‑Run</strong> apuesta por la esencia: correr, saltar, recolectar, sobrevivir. 
    Sin XP, sin niveles, sin skins de colores (hasta que decidamos ponerlas). El sistema de moneda con latas es intuitivo: ves una lata, la quieres, la atrapas. 
    El botón de salir está siempre a mano, la pausa te permite ir al menú sin perder tu carrera. El salto cargado añade una capa de estrategia: ¿salto corto o largo? 
    Todo está pensado para que <em>cualquier persona pueda jugar y divertirse sin leer un manual de 100 páginas.</em> (Aunque este README sea larguísimo, es por puro placer narrativo).
</p>
