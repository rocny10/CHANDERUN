## 🎮 **Control Scheme**

| 🎮 Input | 📍 Context | ⚡ Action |
|----------|------------|-----------|
| `SPACE` (hold) | Gameplay | Charge jump |
| `SPACE` (release) | Gameplay | Execute jump |
| `↓` / `S` | Gameplay | Crouch |
| `ESC` | Gameplay | Pause |
| `ESC` / `Q` | Pause | Resume / Return to menu |
| `T` | Menu | Open shop |
| `1` | Shop | Purchase Second Chance (50 🥤) |
| 🖱️ Click (bottom right) | Gameplay | Pause |
